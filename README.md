# Documentation

This application is completed using ReactJs

## Library Used

React - 18.2.0
React-dom - 18.2.0
Axios - 1.6.8
React-router-dom - 6.28.0

## How to run this application

**Step 1**
Clone the git repo using the command line:
`https://github.com/xridhar/accendo_fe.git`

**Step 2**
executed from the `accendo_fe` folder, ensuring that the **node version is greater than 18**.
`npm install`

**Step 3**
Execute the command in a web browser to view the result: `http://localhost:5173/`
`node run dev`
